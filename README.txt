FEATURES & BENEFITS
-------------------

**IMCE Search features**

Module adds functionality for searching files
from user interface of the 
IMCE file browser (https://www.drupal.org/project/imce).

It also works well with the Linkit module

CONFIGURATION
-------------

Settings page is located at:
 `admin/config/media/imce`
